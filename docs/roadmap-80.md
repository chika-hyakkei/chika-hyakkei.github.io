# メタスコア80点相当ロードマップ

最終更新: 2026-07-22

## 目標

地下百景を「小規模な無料ブラウザRPGとして、欠点は残っても人へ薦められる完成度」まで高める。内部目標は83点、外部プレイテストの中央値78点以上とし、総合的に80点相当を狙う。

## 固定方針

- 100階踏破までの標準所要時間は60〜90分。
- 難易度は「難しいが公平」。死亡時に、次は判断を変えれば進めると感じられることを優先する。
- モンスター図鑑は発見式を維持し、未討伐個体は名前と攻略情報を伏せる。
- アイテムは存在と分類を一覧で確認できるが、詳細効果は入手後に解放する方式へ移行する。
- 100階踏破を終点とし、無限階層は追加しない。
- ガチャ、スタミナ、ログインボーナス、オンライン対戦は80点到達前に追加しない。

## 実装段階

### 第0段階 現状固定と基準計測

- 公開版を基準バージョンとして記録する。
- 開始、死亡、中断、帰還、踏破の状態遷移を整理する。
- 既知の進行不能、黒画面、表示崩れを一覧化する。
- 職業別の到達率、死因、所要時間を現在値として保存する。
- iPhone Safari、Android Chrome、PC Chromeの確認表を作る。

完了条件: 不具合を再現でき、以後の変更前後を同じ条件で比較できる。

### 第1段階 安定性

- 開始、死亡、中断、帰還、踏破の回帰テストを追加する。
- 宝箱、満杯時変換、商店購入、装備交換の回帰テストを追加する。
- 戦闘中、商店中、階移動直後のセーブ復元を検証する。
- ランキング通信失敗でも冒険終了を完了させる。
- 未送信ランキングを端末へ一時保存し、次回接続時に再送する。
- 描画エラー時に真っ黒にならず、再読込またはセーブ復旧を選べる画面を出す。

完了条件: 主要終了処理100回で例外なし。ランキング停止中も終了可能。既存セーブを保持。対象3環境で進行不能なし。

### 第2段階 初見10分

- 初回だけ移動、予告、防御、回復を順番に案内する。
- 押せないボタンには、その場で理由を表示する。
- 状態異常へ名称、色、残りターン、解除方法を表示する。
- PCとスマートフォンで操作案内を出し分ける。

完了条件: 説明なしの5人全員が3階へ到達し、4人以上が予告・防御・回復を理解する。

### 第2.5段階 多言語化の土台

- 画面文言を日本語直書きから翻訳キーへ段階的に分離する。
- セーブ値・職業ID・アイテムID・ランキング値は言語に依存させない。
- 日本語を既定にし、端末言語の自動判定と手動切替を両立する。
- 未翻訳キーは日本語へ戻し、翻訳漏れで進行不能にしない。

完了条件: 日本語版の見た目と挙動を変えず、主要画面を英語へ置換できる構造になる。

### 第3段階 敵と階層主の攻略差

- 25素体へ固有の攻略テーマを持たせる。
- 通常、凶暴、呪染、深淵で行動、弱点、耐性を変える。
- 階層主へHP段階別の行動を実装する。
- 100階ボスで予告、防御、状態回復、装備ビルドを総合的に試す。

完了条件: 敵名と予告を見て行動を変える必要があり、10体の階層主に固有ギミックがある。

### 第4段階 アイテムとビルド

- 一覧、宝箱、商店、装備効果を共通マスターへ統一する。
- 100アイテムすべてを実際に入手・使用可能にする。
- 吸血、防御反撃、会心、MP循環、状態耐性、瀕死強化を成立させる。
- 装備比較と特殊効果の発動表示を追加する。
- アイテム詳細の入手済み管理をMetaへ追加する。

完了条件: 同じ職業で最低3ビルドが成立し、一覧の効果と実際の効果が一致する。

### 第5段階 バランス

- 全職業、複数方針、最低1万固定シードで計測する。
- 10、30、50、70、90、100階の到達率と残資源、死因、所要時間を保存する。
- 初見相当の100階到達率3〜5%、予告活用時10〜15%を目標にする。
- 全職業が50階へ現実的に到達できるよう調整する。

完了条件: 回復不足だけの死亡が過半数にならず、適切な防御で薬なしの通常敵1〜2戦を突破できる。

### 第6段階 絵・音・演出

- 主人公の服装、色、武器サイズを全方向・全行動で統一する。
- 攻撃、防御、魔法、被弾、疲労とHP反映のタイミングを同期する。
- 10階帯ごとの背景、装飾、BGM差を明確にする。
- BGMと効果音の音量、音色、密度を統一する。

完了条件: 誤った行動アニメがなく、HP変化が演出と一致し、全素材が同一作品に見える。

### 第6.5段階 英語版

- タイトル、初見案内、探索、戦闘、商店、図鑑、ランキングを英訳する。
- 英語の文字幅に合わせ、スマートフォン表示とボタン折返しを調整する。
- `/en/` から英語で直接開始できる導線と、ゲーム内言語切替を用意する。
- 英語話者5人以上で、予告・防御・回復の理解と不自然な翻訳を確認する。

完了条件: 英語だけで100階踏破まで操作でき、主要画面に日本語の取り残しがない。

### 第7段階 再挑戦要素

- 熟練度の次の解放内容を表示する。
- 墓と回収候補を開始前に表示する。
- 地下依頼の進捗を探索中に表示する。
- 死亡後に次回の改善点を短く提示する。
- 既存6職業の差別化後に、50階・80階職業を検討する。

### 第8段階 外部テストと公開判定

- 最初は説明なし5人、最終的に15〜20人で確認する。
- 重大な進行不能0件、再挑戦意向70%以上、理不尽死の回答30%未満を合格基準にする。
- 平均75点以上、中央値78点以上を確認する。

## 実装順

安定性 → 初見10分 → 多言語化の土台 → 敵の攻略差 → アイテムとビルド → バランス → 絵・音・演出 → 英語版 → 再挑戦 → 外部テスト。

各段階は、コード、テスト、実機確認、資料更新がすべて完了してから次へ進む。
