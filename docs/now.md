# 現在地

最終更新: 2026-07-21

## 公開中

- ゲーム: `https://chika-hyakkei.github.io/`
- ランキング API: `https://chika-hyakkei-ranking.chika-hyakkei.workers.dev`
- 公開経路: GitHub `main` へのpush → GitHub Actions → GitHub Pages

## 現在の機能

- 100階踏破を目標にしたブラウザローグライク
- 6職業、100体の魔物図鑑、100アイテム、階層主・遺物・墓・熟練度
- 敵行動予告、防御、状態異常、回復薬上限3、爆裂石
- タップ移動、矢印/WASD移動、戦闘ショートカット
- 端末内の匿名テスト記録
- 終了時に自動記録する週間・総合オンラインランキング
- タイトル画面の最新更新要約と週間ランキングTOP3
- 左上のロゴ画像（`public/assets/title-logo.webp`）
- Cloudflare Web Analyticsによる匿名のアクセス集計

## 直近の注意

- ランキングはカジュアル用途。クライアント申告値を受け取るため大会級の不正対策は未実装。
- テストモードだけはランキング対象外。通常プレイの死亡・中断・帰還・踏破は自動記録される。
- `promo/` と `social/` は未追跡の制作素材。明示的な依頼がない限りコミットしない。
