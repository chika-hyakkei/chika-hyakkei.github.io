# ゲーム仕様

最終確認: 2026-07-23

## 目的と進行

- 地下100階の最終ボスを倒すと踏破。
- 10階ごとに階層主が出現し、撃破後に遺物を1つ選ぶ。
- 10階ごとに帰還または継続を選べる。
- 死亡時は冒険中のレベル、装備、持ち物、金貨を失う。図鑑、称号、熟練度、最高記録、墓は残る。

## 戦闘

- 敵の次行動を表示する。通常攻撃、強攻撃、防御、溜め、状態異常がある。
- プレイヤーは攻撃、防御、職業技、回復薬、爆裂石、逃走を選ぶ。
- 防御は大技・状態異常への主な対抗手段。
- 毒、出血、呪い、暗闇、麻痺がある。
- 状態異常は残りターン、効果、解除方法をステータス欄へ表示する。暗闇では攻撃失敗、麻痺では回復薬以外の行動失敗が起こり得る。
- 回復薬は最大3個。開始時1個、最大HPの45%か最低18HPを回復し、使用時も敵の行動が進む。
- 爆裂石は一発逆転用の攻撃アイテムで、所持上限は2個。
- プレイヤーのHP・MPは数値と10分割の宝石ゲージを併記する。端数は最後の区画を部分点灯し、HP半分以下では赤い区画を明滅させる。
- 弱点攻撃は1.35倍、耐性攻撃は0.55倍。敵が「身構える」予告中はプレイヤーの攻撃も軽減される。
- 状態異常予告を防御した場合、付与そのものは受けるが持続を3ターンから1ターンへ抑える。

### 1〜20階の攻略差

- 洞ミミズ：溜めの次に締めつけ。強攻撃を防御する。
- 苔スライム：HP半分で1回だけ分裂回復。半分付近から大技で仕留める。
- 骨ネズミ：通常攻撃が二連噛み。長期戦を避ける。
- 夜コウモリ：与えた傷の35%を吸収する。
- 穴ゴブリン：攻撃時に階層相当の金貨を盗む。
- 岩トカゲ：甲殻防御中は攻撃を18%まで軽減する。
- 亡者の鎧：身構え中に攻撃すると呪いの反撃を受ける。
- 火喰い虫：炎攻撃を吸収して回復し、氷に弱い。
- 影オオカミ：HP25%以下で強攻撃中心へ変化する。
- 深層の目：状態異常予告では暗闇を与える。
- 10階と20階の階層主は固定行動パターンを持ち、HP半分以下で別パターンへ変わる。

### 21〜100階の攻略差

- 泥王子：状態異常予告で暗闇。防御で持続を短縮する。
- 硝子蜘蛛：毒糸で毒。状態異常予告を防御する。
- 空洞兵：盾壁中は攻撃を25%まで軽減する。
- 灯火鬼：強攻撃の火柱がさらに18%強くなる。溜めの後を防御する。
- 鉱脈蛇：攻撃時にMPを奪う。技を早めに使って短期決戦にする。
- 月喰い蛾：状態異常予告で麻痺。
- 墓守犬：状態異常予告で出血。
- 黒潮魚：プレイヤーの守備を45%だけ参照して攻撃する。
- 裂け兎：通常攻撃が三連撃扱いで22%強くなる。
- 鉄殻亀：身構え中に攻撃すると出血の反撃を受ける。
- 奈落蜂：通常攻撃にも一定間隔で麻痺が混じる。
- 鏡面騎士：炎・氷・雷・聖の攻撃を反射する。物理か爆裂石を使う。
- 星屑蛙：身構え・溜めのたび最大HPの8%を再生する。
- 虚ろ司祭：状態異常予告で呪い。
- 深淵の手：プレイヤーHP35%以下では攻撃が30%強くなる。

通常・凶暴・呪染・深淵は同じ素体でも別個体として扱う。凶暴は攻撃・強攻撃中心、呪染は呪い行動、深淵は強攻撃と麻痺行動を加える。弱点と耐性も変異ごとに変わる。21階以降は階層帯に応じて出現可能な素体・変異が広がり、81〜100階では全25素体・4変異が候補になる。

### 階層主

全10体が固有の前半パターンとHP半分以下の後半パターンを持つ。炎吸収、魔法反射、瀕死連撃、再生、暗闇、守備貫通、三連撃、瀕死狩りなど、各区間で学んだ通常敵の判断を複合して試す。戦闘画面では階層主の攻略法、弱点、耐性を最初から表示する。100階は攻撃・防御・状態異常・溜め・強攻撃の5予告をすべて使う。

### 図鑑の情報開示

- 未遭遇：影だけ。
- 遭遇：名前と出現階。
- 1体討伐：弱点と耐性。
- 3体討伐：固有行動と兆候。
- 5体討伐：具体的な攻略法。

## 探索と経済

- 3階ごとに商店。回復薬は各店1個、価格は階層帯により上昇する。
- 階移動時に最大HPの18%とMP3を回復する（遺物で増加可能）。
- 宝箱から金貨、回復薬、爆裂石、装備を得る。持ち物が一杯なら一部は金貨変換される。

## 初回案内

- 通常プレイの初回だけ、移動、敵予告、防御、状態異常、回復を実際の場面で順に案内する。
- スマートフォンではタップと十字キー、PCでは矢印/WASDとクリックを説明する。
- 案内はゲーム操作を止めない小窓とし、1件ずつ閉じられる。まとめて終了でき、フッターから再表示できる。
- MP不足、道具未所持、階層主から逃走不能などは、該当コマンド内に理由を表示する。

## 再挑戦と恒久進行

- 死亡階に墓と遺品喰らいが現れる。倒すと前回の装備・持ち物から1個を回収できる。
- 熟練度は職業の使用で進み、称号・職業などの解放に使う。大きな恒久能力上昇には使わない。
- テストモードは開発確認専用で、通常の匿名テスト記録・オンラインランキングの対象外。
