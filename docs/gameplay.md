# ゲーム仕様

最終確認: 2026-07-19

## 目的と進行

- 地下100階の最終ボスを倒すと踏破。
- 10階ごとに階層主が出現し、撃破後に遺物を1つ選ぶ。
- 10階ごとに帰還または継続を選べる。
- 死亡時は冒険中のレベル、装備、持ち物、金貨を失う。図鑑、称号、熟練度、最高記録、墓は残る。

## 戦闘

- 敵の次行動を表示する。通常攻撃、強攻撃、防御、溜め、状態異常がある。
- プレイヤーは攻撃、防御、職業技、回復薬、爆裂石、逃走を選ぶ。
- 防御は大技・状態異常への主な対抗手段。
- 毒、出血、呪い、暗闇、麻痺がある。
- 回復薬は最大3個。開始時1個、最大HPの45%か最低18HPを回復し、使用時も敵の行動が進む。
- 爆裂石は一発逆転用の攻撃アイテムで、所持上限は2個。

## 探索と経済

- 3階ごとに商店。回復薬は各店1個、価格は階層帯により上昇する。
- 階移動時に最大HPの18%とMP3を回復する（遺物で増加可能）。
- 宝箱から金貨、回復薬、爆裂石、装備を得る。持ち物が一杯なら一部は金貨変換される。

## 再挑戦と恒久進行

- 死亡階に墓と遺品喰らいが現れる。倒すと前回の装備・持ち物から1個を回収できる。
- 熟練度は職業の使用で進み、称号・職業などの解放に使う。大きな恒久能力上昇には使わない。
- テストモードは開発確認専用で、通常の匿名テスト記録・オンラインランキングの対象外。
