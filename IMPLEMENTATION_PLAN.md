# 地下百景 — 100階拡張・確定実装仕様

最終目標は、地下100階の最終ボス撃破を明確なクリア条件とするローグライクRPGです。死亡時はレベル、装備、アイテム、金貨を失い、図鑑・称号・職業熟練度・最高記録・墓データのみ端末に保存します。スタミナ、ログインボーナス、ガチャは導入しません。

## 確定ルール

- 100階で終了。無限階層は追加しない。
- 敵は通常攻撃・大技・防御・回復・状態異常を行い、次の行動を常に予告する。
- 通常敵は2〜4手、階層主はHPに応じた複数段階の行動パターンを持つ。
- 状態異常は毒、出血、呪い、暗闇、麻痺。解除手段も用意する。
- 防御は被ダメージを大幅に抑え、大技と一部の状態異常への対抗手段とする。
- 回復薬は開始時1個、上限3個。最大HPの45%（最低18HP）を回復し、使用時も敵の行動は進む。
- 商店は3階ごと、回復薬は各店1個。価格は12〜48G。宝箱からの初期出現率は約24%。満杯時の回復薬は少額の金貨に変換する。
- 階移動時は最大HPの18%とMP3を回復する。
- 10階ごとに帰還／継続を選べる。継続で得点倍率と希少品率を上昇させる。
- 100階クリアで専用エンディング、称号、職業色を解放する。

## コンテンツ目標

- 敵：25素体×4変異（通常・凶暴・呪染・深淵）を基礎に、階層主10体を含む図鑑100体。
- アイテム：武器30、防具30、装飾20、消耗品10、遺物10の計100個。
- 装備効果：吸血、会心、反撃、MP回復、状態耐性、瀕死強化など。
- 階層主撃破時：遺物3択から1個を選び、冒険ごとのビルドを作る。
- 墓：死亡階に墓と遺品喰らいを生成。倒すと失った物を1個だけ回収でき、再死亡で古い墓は置換される。
- 恒久進行：職業熟練度で初期装備、技、外見色、称号を解放。ただし大幅な恒久ステータス上昇は行わない。
- 冒険開始時：3件の地下依頼から1件を選択。日替わり固定迷宮と共有コードに拡張可能なデータ構造にする。

## 100階構成

| 階層 | 主題 |
| --- | --- |
| 1〜10 | 基本操作と単純な予告 |
| 11〜30 | 状態異常と弱点 |
| 31〜60 | 属性、連撃、敵の回復 |
| 61〜90 | 複合行動、耐性、危険な状態異常 |
| 91〜100 | 完成した装備ビルドと判断力 |

各10階で環境、敵、商店装備、BGMを切り替え、10階ごとに階層主を配置します。100階には固有行動と演出を持つ最終ボスを置きます。

## 保存と容量

- Run：状態異常、敵予告、遺物、商店在庫、依頼、墓回収状態を追加。
- Meta：図鑑、熟練度、称号、墓、100階クリア記録を追加。
- セーブをv2へ移行し、既存の保存データには不足項目を補完。
- モンスターとアイテムをデータ配列に分離。
- CSSドット部品とWeb Audioを使用し、大量の画像・音声を追加しない。
- 公開ビルドは5MB以内、OG画像を除く初回転送は約1MBを目標とする。

## 難易度検証

- 全職業・複数装備構成・固定シード1万種類以上で自動戦闘／探索を検証する。
- 10、30、50、70、90、100階で到達率、HP、MP、薬、死亡原因を計測する。
- 初見相当の行動で100階到達率3〜5%、予告を適切に使う行動で10〜15%を目標にする。
- 適切な防御と技なら、回復薬なしで通常敵1〜2戦を突破できることを基準とする。
- 各10階区間に平均2〜4回の回復機会を確保し、商店では薬と装備の購入を迷える金貨量にする。
- 不足時は、通常敵攻撃力、薬の回復量、宝箱の薬率、商店価格の順に調整する。薬上限3個・店ごと1個は維持する。
- 説明なしの5人以上による試遊で「難しいが次は進めそう」を確認する。
